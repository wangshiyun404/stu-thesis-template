# 汕头大学研究生学位论文 Word 模板

v1.0.0版本弃用即可，直接使用最新的v2.0.0版本，这是一个用于汕头大学研究生学位论文写作的 Word 模板仓库，包含可直接填写的 `.docx` 模板文件和`.dotx` 模板文件（两者仅文件名后缀不同）。

## 文件说明

- `汕头大学学位论文word版参考模板v2.0.0[2026.3.25修订].docx`  
  汕头大学研究生学位论文 Word 模板。

## 使用方法

1. 下载本仓库中的 `.docx` 模板文件。
2. 打开后建议先“另存为”自己的论文文件。
3. 使用模板内预设的标题样式撰写论文。
4. 目录、图目录、表目录可在完成后自动更新。

## 说明

- 本模板适用于 Word 环境下的学位论文写作。
- 使用前建议仔细核对学校当年的最新格式要求。
- 若模板后续修订，可直接在本仓库更新版本。
- 第二个v2.0.0版本的.dotx后缀的模板属于原始模板（.dotx 是 Word 的模板文件格式，主要用于生成新的文档。通常情况下，用户打开 .dotx 文件时，系统会基于该模板新建一个文档，后续可另存为 .docx 文件并继续编辑，而原模板文件一般会保留在原位置不变。与普通文档相比，.dotx 更有利于统一版式和规范文档格式，但其兼容性较高，虽然也有一定可能受到不同版本的 Word、WPS 及具体功能支持情况的影响）。如果使用的是最新版本的office的word软件，不存在乱码、不兼容等问题，则使用.docx后缀的模板就够了，不用理会.dotx的模板。
