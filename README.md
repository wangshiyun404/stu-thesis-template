# 汕头大学研究生学位论文 Word 模板

这是一个用于汕头大学研究生学位论文写作的 Word 模板仓库，包含可直接填写的 `.docx` 模板文件。

## 文件说明

- `汕头大学学位论文word版参考模板v2.0.0[2026.3.25修订].docx`  
  汕头大学研究生学位论文 Word 模板。

## 使用方法

1. 下载本仓库中的 `.docx` 模板文件。
2. 打开后建议先“另存为”自己的论文文件。
3. 使用模板内预设的标题样式撰写论文。
4. 目录、图目录、表目录可在完成后自动更新。

## 说明

- 本模板适用于 Word 环境下的学位论文写作。
- 使用前建议仔细核对学校当年的最新格式要求。
- 若模板后续修订，可直接在本仓库更新版本。
- 第二个v2.0.0版本的.dotx后缀的模板属于原始模板（永远不会受到word或wps软件版本的限制，不会出现不兼容等各种各样的问题），每次打开都会生成一个新文件，另存为.docx文件即可，然后继续编辑.docx文件即可，而.dotx文件仍然会在原来的位置不动。如果使用的是最新版本的office的word软件，不存在乱码、不兼容等问题，则使用.docx后缀的模板就够了，不用理会.dotx的模板。
